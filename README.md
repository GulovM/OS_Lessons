# OS_Lessons
Here are the assigments for the subject OS

## First task (Project1.cpp):
1. Create a vector of a million elements. Fill it with random numbers.

2. Calculate the sum of this vector on a single thread. That is, simply in a loop, as usual.

3. Measure the program's execution time.

4. Calculate the sum of the vector in parts, placing each part in a separate thread.

5. Measure the program's execution time and compare it with the previous one.
