# Первое задание (Project1.cpp):
1. Создать вектор из миллиона элементов и заполнить его случайными числами.
2. Вычислить сумму этого вектора в одном потоке, то есть просто в обычном цикле.
3. Измерить время выполнения программы.
4. Вычислить сумму вектора по частям, поместив каждую часть в отдельный поток.
5. Измерить время выполнения программы и сравнить его с предыдущим результатом.

## Вывод: 
* Sum (single thread): 50510116
* Time (1 thread): 0.0025376 s
* Sum (4 threads): 50510116
* Time (4 threads): 0.0069016 s
* Speedup: 0.367683x
